## 简单的使用方法：

### 克隆项目到本地
	git clon git@gitee.com:Tangkz/tang-long.git	

	
使用`pip`安装第三方依赖

	pip install django
	
修改`settings.example.py`文件为`settings.py`

运行`migrate`命令，创建数据库和数据表

	python manage.py migrate
	
启动服务器

	python manage.py runserver
